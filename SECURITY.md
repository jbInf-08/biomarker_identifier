# Security Policy

## Supported Versions

Security fixes are provided for the latest `main` branch state and the latest release branch (if published).

## Reporting a Vulnerability

Please do not open public issues for undisclosed vulnerabilities.

- Email: `security@jbinf-08.dev`
- Include: affected component, reproduction steps, impact, and any suggested fix.
- Expected response: acknowledgment within 72 hours, triage update within 7 days.

## Disclosure Process

1. Report received and triaged.
2. Maintainer validates and assigns severity.
3. Patch is prepared and tested.
4. Coordinated disclosure note is published after fix release.
