## Mobile Store Assets Checklist

Before App Store / Play submission:

- [ ] Final app icon (1024x1024) and adaptive icon variants
- [ ] Splash screen assets for light/dark
- [ ] Store screenshots (phone + tablet where required)
- [ ] App preview video (recommended)
- [ ] Privacy policy URL and support URL
- [ ] Data safety / nutrition labels completed
- [ ] Sign-in and tenant switching UX reviewed
- [ ] Push notification permission rationale text
- [ ] Build numbers incremented (`app.json`, EAS profile)
- [ ] TestFlight / internal track smoke test complete
